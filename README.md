# DriveGo

Nền tảng học lái xe và luyện thi bằng lái — giao diện React + Vite + Tailwind CSS v4.

## Chạy local

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — dev server
- `npm run build` — build production
- `npm run lint` — ESLint
- `npm run preview` — xem bản build
